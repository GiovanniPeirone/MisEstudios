


Breack
-------------------------------------------------------------------------------------------------------------

El breack sirbe para romper un bucle

for (int i = 0; i < 10; i++) {  
  if (i == 4) {  
    break;  
  }  
  cout << i << "\n";  
}


Continue
-------------------------------------------------------------------------------------------------------------

El continue sirbe para romper el bucle si una condicion se cumple


for (int i = 0; i < 10; i++) {
    if (i == 4) {
      continue;
    }
    cout << i << "\n";
  }
